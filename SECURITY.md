# Security Policy

Consulting-as-Code™ is a strategic reasoning system.

While this repository does not contain executable code,
security, integrity, and misuse concerns are taken seriously.

## Reporting Concerns
If you identify:
- Misuse of the Consulting-as-Code™ term
- Misrepresentation of the architecture
- Security implications of described systems

Please contact:
hello@vuduvations.io

Do not open public issues for sensitive concerns.
